How to play

First, find a friend who can share your keyboard, then fight!

The goal depends on who you're playing as. For "Mr. Red Overalls" you're trying to collect as many mushrooms as possible, but watch out for the poison ones! For "Dino King" you want Overalls to suffer as much as possible.

In order to be able to spawn mushrooms as Dino King, there's a certain unassuming item that you must collect.

About

I remixed this classic game for a "sell out"-themed art show. Based on Meth Meth Method's Super Mario in a Browser tutorial code, I refactored it to follow an ECS pattern, which made it easier for me to expiriment with changes to gameplay. I plan to use this code as a template for more original browser-based pixel art platformer games.

Source code

The source code, though at the moment somewhat disorganized and full of commented-out sections due to the time constraints under which it was made, can be found on Github.
