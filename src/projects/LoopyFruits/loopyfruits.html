How to play

First, find a friend with their own desktop or laptop computer, then open loopy fruits and share the URL with them. More instructions can be found in the "FAQ."

Notes

This is a project I was working on for a few months in 2022. It started with the modest goal of having a few simple affordances, using common personal computer hardware, for playing music with friends in a virtual jam room. After achieving that, I too quickly expanded the scope to include simulating analog instruments, including chords, starting with the guitar. I was curious about Elm so that's what I used to write most of the front-end, but as that code grew in complexity I found that I was spending a lot of time thinking about how to work around limitations in Elm's type system and it began to feel unsustainable. Around the same time, I started working on a project I was being paid for, so LoopyFruits fell by the wayside. I'm planning to pick it back up again soon, and will probably start with migrating the front-end to something like PureScript, since I have a lot of ideas to improve the UX anyway.
